# Phoenix Biodiversity Change  
***
**Author:** William Mullins | **Date:** 12-06-2025

## About  
Phoenix is one of the fastest-growing areas in the United States. This repository quantifies how that growth has altered terrestrial biodiversity using the **Biodiversity Intactness Index (BII)** at 100 meter resolution.  

## Repository Structure  
``phoenix-biodiversity/
├── README.md
├── Phoenix_Biodiversity.ipynb

## Data  This analysis utilizes two key datasets:

1.  **Phoenix Subdivision Shapefile**: A shapefile containing county subdivisions in Arizona as of 2025 from the U.S. Census Bureau. It can be accessed [here](https://catalog.data.gov/dataset/tiger-line-shapefile-current-state-arizona-county-subdivision).
2.  **Biodiversity Intactness Index (BII) Time Series**: A dataset containing estimates of terrestrial biodiversity intactness as 100-meter maps from **2017 to 2020**. It can be accessed [here](https://planetarycomputer.microsoft.com/dataset/io-biodiversity).
